# pgeneral-C
Práctica general de la asignatura de compilación.

La práctica consiste en construir el front-end de un compilador utilizando la técnica de construcción de traductores ascendente, a partir de un esquema de traducción dirigida por la sintaxis. El lenguajede entrada (fuente) al compilador es un lenguaje de alto nivel, y el de salida un código de tresdirecciones. Se ha realizado la implementación del traductor en C++, Flex y Bison.
